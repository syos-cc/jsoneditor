README / Help

Overview

This page is a standalone JSON editor. You can load a JSON file, edit it visually as a structured diagram, and download it again.

Upload

  • Use Upload to open a local .json file.
  • The file is processed entirely in your browser and is never uploaded.

Download

  • Use Download to save the current document as edited.json.
  • String, Number, Boolean, Array and Object are exported as native JSON types.

Adding Elements

  • Add String – adds a string value.
  • Add Number – adds a numeric value.
  • Add Boolean – adds a boolean value.
  • Add Array – adds an array.
  • Add Object – adds an object.

Drag a new element from the toolbar into an Object or Array.

Editing

  • Click a field to edit its key or value.
  • Object entries are displayed as Key: Value.
  • Array elements have no key.
  • For Boolean values, only true evaluates to true; every other value becomes false.

Moving

  • Existing elements can be moved using drag & drop.
  • Moving an element into an Array removes its key.
  • Moving an element into an Object automatically creates a key if needed.

Deleting

  • Every element except the root object has a × button.
  • The selected element and all of its children are removed.

Path Display

  • Click any element to display its path at the top of the page.
  • The path format follows jq, for example .F.J[1].
  • . represents the root object.

Notes

  • No server is required. The editor runs directly as a local HTML file.
  • Currently null values are imported as empty strings.
  • Undo/Redo is not implemented yet.
